# Proyecto Final MODULO 1

Este proyecto es el desarrollo de una página web básica con HTML en GitHub pages.


# Características

El proyecto es simple y consta de un solo archivo HTML que contiene la crítica de una película. imágenes de referencia y un enlace para visualizarla desde un  servicio de streaming.

 **Características Técnicas**
En este proyecto se usan etiquetas fundamentales como: *h1, p, ul, form, img.*
